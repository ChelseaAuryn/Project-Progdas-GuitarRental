Deskripsi Singkat : 
Aplikasi peminjaman gitar adalah sebuah platform yang dirancang untuk memudahkan pengguna menyewa gitar dengan cara yang lebih praktis. Pengguna dapat melihat daftar gitar, memilih gitar yang akan disewa, menyewa gitar, mengembalikan gitar, melihat riwayat rental dan sebagai admin dapat melihat riwayat gitar yang disewa oleh pengguna. Aplikasi ini juga dirancang dengan pendekatan berfokus pada objek, di mana setiap komponen seperti gitar, penyewa, dan transaksi dibuat sebagai class.
